# Solar Tracking System (DLD Project)

## Description
This project is a Digital Logic Design (DLD) implementation of an automatic solar tracking system using Arduino, LDR sensors, and a servo motor. It detects light intensity and adjusts the servo position to maximize solar exposure.

## Components Used
- Arduino UNO  
- LDR (Photoresistor)  
- Servo Motor (SG90)  
- Resistors (2.2kΩ)  
- Breadboard & Jumper Wires  

## Working Principle
The LDR senses light intensity and sends analog values to Arduino. The Arduino processes this data and controls the servo motor to align toward the strongest light source.

## Features

- Real-time servo movement  
- Cost-effective prototype  
- Simple embedded system design
